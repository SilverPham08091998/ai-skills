# AI Skills

Repository này dùng để lưu trữ các skill, guideline và rule set dành cho nhiều AI coding assistant như Claude, Codex, Cursor và các công cụ AI khác.

Mục tiêu chính là chuẩn hóa cách AI hỗ trợ phát triển phần mềm theo từng vai trò, công nghệ và ngữ cảnh dự án.

## Mục Đích

- Xây dựng bộ guideline có thể tái sử dụng cho các AI assistant.
- Chia nhỏ kiến thức theo domain, ví dụ mobile engineering, React Native, architecture, state management.
- Giúp AI phản hồi nhất quán hơn khi review code, thiết kế kiến trúc, viết feature hoặc refactor.
- Dễ dàng mở rộng để hỗ trợ thêm Claude, Codex, Cursor hoặc các agent khác.

## Cấu Trúc Hiện Tại

```text
claude/
  guidelines/
    mobile-engineer/
      architecture/
      foundation/
      react-native/
      state-management/
```

## Hướng Phát Triển

- Thêm guideline riêng cho Codex.
- Thêm rule set riêng cho Cursor.
- Chuẩn hóa format skill để có thể copy hoặc import vào từng AI tool.
- Bổ sung ví dụ prompt và use case thực tế cho từng nhóm guideline.

## Ghi Chú

Repo này không phải source code của một ứng dụng cụ thể. Đây là nơi quản lý tri thức, nguyên tắc và hướng dẫn để các AI assistant làm việc hiệu quả và nhất quán hơn trong quá trình phát triển phần mềm.
